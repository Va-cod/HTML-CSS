# 📦 Inventory Manager

## 📝 Description
This is a simple inventory system in Python.  
It helps to manage products with name, price, and quantity.

## ⚙️ Features
- 📋 Show all products in the inventory
- ➕ Add new products
- 🔍 Search a product by name
- ✏️ Update product information
- ❌ Delete a product
- 📊 Calculate statistics (total value, most expensive product, more quantity)
- 💾 Save inventory to a CSV file
- 📂 Load inventory from a CSV file

## ⚡ How it works
- Each product has:
  - 🏷️ Name
  - 💲 Price
  - 🔢 Quantity

- The system stores products in a list of dictionaries.

## 📁 CSV Functions
- 💾 The system can save the inventory in a CSV file.
- 📂 The system can load products from a CSV file.
- ⚠️ It checks errors in the file.
- 🚫 It ignores invalid rows.
- You can choose:
  - 🔄 Replace the inventory
  - 🔗 Merge the data

## 🧰 Requirements
- 🐍 Python 3

## ▶️ How to use
1. ▶️ Run the program.
2. ⚙️ Use the functions to manage products.
3. 💾 Save or 📂 load data when needed.
