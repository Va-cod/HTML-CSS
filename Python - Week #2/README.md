# 📦 Inventory System

📖 **Description**

This is a simple program in Python.
It helps to manage products in an inventory.


⚙️ **Functions**

Show all products

Add new products

Calculate total inventory value

Show number of products


▶️ **How to Use**

Run the program and choose an option from the menu:

1. Display inventory

2. Add products

3. Calculate statistics

4. Log out


⚠️ **Notes**

Price must be a positive number

Quantity must be a positive number
